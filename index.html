Unix commands

Aleksandr Kuralenko 2025.

Let's start with the (ls-command).


								bash
		
								$ ls
								


								bash
			
								$ file.txt  Documents  Downloads  image.jpg  report.pdf

More detail information use: 


							bash
		
							$ ls -l
								


							bash
	
							-rw-r--r-- 1 user group  4096 Dec  5 10:30 file.txt
							drwxr-xr-x 2 user group  4096 Dec  5 09:15 Documents
						


							bash
		
							$ ls -a
								


							bash
	
							$ .  ..  .bashrc  file.txt  Documents  .gitignore  Downloads
						


							bash
	
							$ ls -la

Let's move on to the (cd - change directory) command


								bash
		
								$ cd Documents
							


								bash
		
								$ cd or cd ~  /* for back to home directory */
							


								bash
		
								$ cd . or cd ..

pwd - Print Working Directory


								bash
		
								$ pwd
							


								bash
		
								$ pwd
								/home/username

Copy files (cp - copy)


								bash
		
								$ cp report.txt Backup/
							


								bash
		
								$ cp -r Folder1 Folder2/

Move files (move/rename)


								bash
		
								$ mv oldname.txt newname.txt
							


								bash
		
								$ mv file.txt ~/Documents/

Remove files (rm - remove)


								bash
		
								$ rm file.txt
							


								bash
		
								$ rm -r OldFolder
The rm command permanently deletes data!

Create directory.


								bash
		
								$ mkdir Folder1
							


								bash
		
								$ rmdir OldFolder

Viewing File Contents (cat - command).


								bash
		
								$ cat report.txt
							


								bash
		
								$ less largefile.log

Creating new files (touch - command).


								bash
		
								$ touch newfile.txt

Search for files - (find command).


								bash
		
								$ find . -name "*.txt"
							


								bash
		
								$ grep "error" file.txt
							


								bash
		
								$ grep -r "error" .

Working with processes — (ps - command).


								bash
		
								$ ps aux
							


								bash
		
								$ kill processPID
							


								bash
		
								$ kill -9 processPID

Output of text or variables — (echo command).


								bash
		
								$ echo "Hello, Unix!"

Get help — (man command).


								bash
		
								$ man ls
